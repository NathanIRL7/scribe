#  Scribe

> Stop writing emails. Start closing clients.

Scribe is an AI-powered tool for freelancers that writes 
personalized cold emails and finds potential clients automatically – 
so you can focus on actual work.

- ✅ Project structure
- ✅ Core functionality
- ✅ Design & UI
